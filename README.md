\# LLM File Assistant



\## Overview



This project demonstrates file system tool calling for resume management.



The assistant can:



\- Read resume files

\- List resume files

\- Search resumes by keyword

\- Create summary files



\## Project Structure



LLM\_File\_Assistant/

|

|-- resumes/

|-- fs\_tools.py

|-- llm\_file\_assistant.py

|-- requirements.txt

|-- README.md



\## Installation



pip install -r requirements.txt



\## Run



python llm\_file\_assistant.py



\## Example Queries



read all resumes



find python resumes



create summary
